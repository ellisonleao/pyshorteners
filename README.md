A simple URL shortening Python Lib, implementing the most famous shorteners.

# Installing

    pip install pyshorteners

# Testing

	make test

# Usage

## Simple example

```python
import pyshorteners

s = pyshorteners.Shortener()
print(s.tinyurl.short('www.google.com'))
# prints 'http://tinyurl.com/HASH'
```
